# Lecture1 强化学习基础

## 强化学习基本概念

### 核心名词

| 概念名词 | 数学表达            | 含义                                           |
| -------- | ------------------- | ---------------------------------------------- |
| 状态     | State   /  S        | 当前时刻环境的概括                             |
| 状态空间 | State Space  / $ S$ | 所有可能的状态集合                             |
| 动作     | action              | 智能体基于当前环境作出的决策                   |
| 动作空间 | action space / $ A$ | 所有可能的动作集合                             |
| 奖励     | reward              | 智能体在执行一个动作后，环境给智能体的一个数值 |



### 基础概念

####  状态转移函数

| 概念名词     | 数学表达          | 含义                                                       |
| ------------ | ----------------- | ---------------------------------------------------------- |
| 状态转移     | state transition  | agent从当前时刻t的状态s转移到下一个状态的过程              |
| 状态转移函数 | $p_t(s^{'|s,a}) $ | $p_t(s^{'|s,a})=\mathbb{P}(S^{'}_{t+1}=s^{'}|S_t=s,A_t=a)$ |

状态转移可以是确定（确定状态转移），也可以是随机的（随机状态转移）

实际中，通常**假设状态转移概率函数是平稳的**

一般写作 $p(s^{'}|s,a)$ 或 $\tau(s,a)$

<font color=red>状态的随机性来源于状态转移</font>

#### 

| 概念名词           | 数学表达 | 含义                                         |
| ------------------ | -------- | -------------------------------------------- |
| 策略（policy）     |          | 如何根据观测到的状态作出决策                 |
| 回报（return）     |          | 从当前时刻开始到本回合结束所有奖励的总和     |
| 轨迹（trajectory） |          | 一个回合中智能体观测到的所有状态、动作、奖励 |

#### 策略

* 随机性策略
  * $\pi:\mathcal {S\cross A \mapsto}[0,1]$
  * 随机抽样，概率动作
* 确定性策略
  * $\mu:\mathcal S \mapsto \mathcal A$
  * 将s作为输入，直接输出动作a而非概率值

<font color=red>动作的随机性来源于策略</font>
